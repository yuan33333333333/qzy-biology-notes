## 一、 什么是植物激素？
>**由植物体内产生**，能从产生部位运送到作用部位，对植物的生长发育有显著影响的微量有机物，叫作植物激素（phytohormone）。

**植物激素（Plant Hormones / Phytohormones）**是指在植物体内合成的、通常从合成部位运送到作用部位，对植物的生长发育产生显著调节作用的**微量有机物**。

- **来源：** 植物体自身产生（内源性）。
    
- **特点：** 含量极微，但“四两拨千斤”，能控制植物的萌发、生长、开花、结果及衰老。
    
- **常见“五大类”：** 生长素、赤霉素、细胞分裂素、脱落酸、乙烯。
    

## 二、 什么是植物生长调节剂？
>人们在多年的研究和实践中，发现一些人工合成的化学物质，具有与 植物激素相似的生理效应。这些**由人工合成的**，对植物的 生长、发育有调节作用的化学物质，称为植物生长调节剂。

**植物生长调节剂（Plant Growth Regulators, 简称 PGRs）**是人工合成的（或从微生物中提取的），具有与天然植物激素相似生理和生物学效应的**组分或化合物**。

- **来源：** 人工化学合成或工厂化发酵（外源性）。
    
- **特点：** 原理是模仿天然激素，但分子结构可能与天然激素相同，也可能完全不同（只要功效类似就行）。
    
- **常见例子：** 萘乙酸（模仿生长素）、矮壮素（抑制生长）、乙烯利（催熟，释放乙烯）。
    

## 三、 它们之间的“两相同、三不同”

要准确辨析这两者，可以通过下面这个表格一眼看清：

| **辨析维度**  | **植物激素（天然）**                                | **植物生长调节剂（人工）**         |
| --------- | ------------------------------------------- | ----------------------- |
| **主要来源**  | 植物体内自行合成                                    | 工业人工化学合成                |
| **提取/生产** | 植物内含量极低，极难大规模提取                             | 可工业化大批量生产，成本低           |
| **稳定性**   | 在植物体内易被相应的酶分解，作用时间短                         | 在植物体内不易被迅速降解，效果更持久、更强烈  |
| **应用效果**  | 效果温和，受植物自身精准调控                              | 效果显著，但对浓度极其敏感（用多容易适得其反） |
| **相同点 1** | **生理功能相同**：都能在极低的浓度下，对植物的生长发育起到促进、抑制或改变的作用。 |                         |
| **相同点 2** | **作用机制相似**：都是通过绑定植物细胞的特定受体，来启动一系列信号传导。      |                         |
