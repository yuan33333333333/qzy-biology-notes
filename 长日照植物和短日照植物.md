## 长日照植物 (Long-day Plants)

长日照植物是指在生长发育过程中，需要**日照时间超过一定限度**（通常为12~14小时以上）**才能开花的植物**。如果光照时间不够，它们就会一直“装傻”只长叶子不开花。

- **核心特点：** **光照时间越长，开花越早**。**如果给予连续的黑暗，它们的开花就会受到抑制**。
    
- **常见代表：** * **农作物：** 小麦、大麦、油菜。
    
    - **蔬菜：** 菠菜、萝卜、白菜。
        
    - **花卉：** 鸢尾、天竺葵。
        
- **开花季节：** 通常在**春末夏初**开花，因为那时候北半球的白天越来越长。
    

## 短日照植物 (Short-day Plants)

短日照植物则恰恰相反，它们需要**日照时间短于一定限度**（通常在12-14小时以下）才能开花。**在长日照条件下，它们会只进行营养生长（长茎叶），而无法进入生殖生长（开花）。**

- **核心特点：** **它们对连续的黑暗（暗期）非常敏感**。==**如果黑夜的时间不够长，或者在黑夜中被一道闪光短暂打断（光闪），它们就无法开花。因此，准确来说它们其实是“长夜植物”。**==
    
- **常见代表：**
    
    - **农作物：** 水稻、玉米、大豆、高粱。
        
    - **花卉/其他：** 菊花、牵牛花、一品红、棉花。
        
- **开花季节：** 通常在**晚夏或秋季**开花，此时白天变短，黑夜变长。

| **特征**      | **长日照植物**                 | **短日照植物**                 |
| ----------- | ------------------------- | ------------------------- |
| **开花要求的日照** | 必须**长于**临界日长（通常 >12-14小时） | 必须**短于**临界日长（通常 <12-14小时） |
| **决定性因素**   | 连续的白昼时间足够长                | **==连续==的黑暗时间（暗期）足够长**    |
| **主要开花季节**  | 春末、夏季                     | 秋季、冬季                     |
| **典型代表**    | 小麦、菠菜、萝卜、油菜               | 菊花、水稻、大豆、玉米               |
