## 1. 种间关系全景拆解

### 🤝 双赢关系（+ / +）

#### ① 原始合作（Protocooperation / 互惠）

- **特点**：两种生物共同生活时，**双方都受益**；但如果分开，双方也能独立生存。这是一种“非必要”的合作。
    
- **实例**：
    
    - **犀牛与犀牛鸟**：犀牛鸟帮犀牛吃掉皮肤上的寄生虫（犀牛受益），同时填饱肚子（犀牛鸟受益）；但它们分开也能活。
        
    - **海葵与寄居蟹**：海葵保护寄居蟹，寄居蟹的移动带海葵去更多地方觅食。
        
- **数学模型/曲线**：由于分开也能活，两条种群数量曲线的波动方向大体一致，但没有严格的绑定关系。
    

#### ② 互利共生（Mutualism）

- **特点**：两种生物共同生活，**双方受益**。由于结合得过于紧密，它们已经“生死相依”，**如果分开，双方或一方无法生存（专性共生）**。
    
- **实例**：
    
    - **豆科植物与根瘤菌**：根瘤菌固定氮素给植物，植物提供有机物。
        
    - **地衣（真菌与藻类）**：真菌提供水和无机盐，藻类进行光合作用提供有机物。
        
    - **大肠杆菌与人类**：大肠杆菌在肠道内帮人类合成维生素，人类提供食物和生存环境。
        
- **数学模型/曲线**：**“同生共死”**。两条曲线同步增加、同步减少。
    

### ⚔️ 双输关系（- / -）

#### ③ 种间竞争（Competition）

- **特点**：两种或两种以上生物相互争夺**相同资源**（如食物、空间、阳光等）。竞争的结果通常是“两败俱伤”（即使赢了也会消耗能量），弱势一方可能被淘汰或被迫改变生态位。
    
- **实例**：
    
    - **大草履虫和双小核草履虫**：在同一培养基中，双小核草履虫胜出，大草履虫死亡。
        
    - **水稻和杂草**：争夺阳光和土壤中的营养。
        
- **数学模型/曲线**：**“你死我活”**或**“双输波动”**。资源有限时，一方数量增加，另一方往往减少。最典型的是“强者恒强，弱者灭绝”的曲线。
    

### 🎯 损人利己关系（+ / -）

#### ④ 捕食（Predation）

- **特点**：一种生物（捕食者）以另一种生物（被捕食者）为食。捕食者通常体型较大或具有攻击性，直接杀死并消耗被捕食者。
    
- **实例**：狼吃羊、猫头鹰捕食田鼠、青蛙吃昆虫。
    
- **数学模型/曲线**：**“先增加者先减少，后增加者后减少”**。两条曲线呈现非同步的周期性波动，捕食者的峰值通常滞后于被捕食者，且捕食者的基数（数量）通常低于被捕食者。![[Pasted image 20260613161322.png]]
    

#### ⑤ 寄生（Parasitism）

- **特点**：一种生物（寄生生物）生活在另一种生物（宿主/寄主）的体内或体表，**从宿主身上摄取营养**。寄生者通常体型小、数量多，一般**不会立刻致死**宿主（因为宿主死了，寄生者也失去食物源）。
    
- **实例**：
    
    - **体内寄生**：蛔虫、猪肉绦虫寄生在人体内；病毒寄生在宿主细胞内。
        
    - **体表寄生**：跳蚤、蜱虫吸食动物血液。
        
    - **植物寄生**：菟丝子寄生在其他植物上吸取养分。
        
- **数学模型/曲线**：宿主数量增加，寄生者随之大增；如果宿主大量死亡，寄生者也会随之减少。但与捕食不同的是，寄生者的数量（个体数）往往远多于宿主。
    

## 2. 种间关系的判定与区分核心

在面对具体案例或坐标曲线时，可以按照以下逻辑链进行判断：

```
                   双方都获利 (+/+)？
                  /              \
                是                否
               /                    \
    分开后还能活？                 一方获利，一方受损 (+/-)？
     /       \                     /                \
   是        否                   是                 否 (双方受损 -/-)
   ↓         ↓                   /    \                     ↓
原始合作   互利共生        直接吃掉？     吸血/吸营养？         种间竞争
                        /         \     \             (争夺相同资源)
                       是          否     宿主通常不死
                       ↓           ↓          ↓
                     捕食         (看情况)    寄生
```

### 📊 核心区分速查表

|**关系类型**|**能量流动方向**|**数量曲线特征**|**核心判定点**|
|---|---|---|---|
|**原始合作**|双向获利|大致同步，分开不影响生存|“锦上添花”，分开也能活|
|**互利共生**|双向获利|严格的**“同生共死”**，曲线起点相同|“生死相依”，分开即死|
|**种间竞争**|负面抑制|**“你死我活”**，一方崛起一方灭绝/剧减|争夺**同一种资源**（空间/食物）|
|**捕食**|被捕食者 → 捕食者|**“后增后减”**，波动有相位差，被捕食者数量多|**直接吃掉**，捕食者体型通常较大|
|**寄生**|宿主 → 寄生者|受宿主基数限制，但寄生者个体数可远超宿主|**吸血/吸营养**，寄生者体型小，宿主通常不死|

## 3. 常见易错点与避坑指南

### ⚠️ 易错点一：分不清“种内”还是“种间”

- **坑点**：看到“大鱼吃小鱼”或“公鹿争夺配偶”就选捕食或竞争。
    
- **纠正**：种间关系必须发生在**不同物种**之间。
    
    - 大鱼吃同种的小鱼（如鲈鱼吃自己的幼鱼），属于**种内捕食（种内共生/种内斗争）**，不能叫捕食。
        
    - 两只公鹿争夺配偶，属于**种内斗争**；而鹿和牛争夺草场，才叫**种间竞争**。
        

### ⚠️ 易错点二：误把“捕食”当成“竞争”

- **坑点**：认为捕食者和被捕食者是在“竞争生存机会”。
    
- **纠正**：竞争是争夺**第三方资源**（如两个人都抢一个面包）；而捕食是**一方把另一方当成资源**（一个人把另一个人吃了）。
    

### ⚠️ 易错点三：混淆“寄生”与“捕食”

- **坑点**：虫子吃植物（如蚜虫吸食植物汁液，或者毛毛虫吃树叶），到底是捕食还是寄生？
    
- **纠正**：
    
    - 如果是一口一口把植物器官吃掉、致死的（如蝗虫过境、毛毛虫吃光叶片），算作**捕食**（广义上的植食作用）。
        
    - 如果是定居在植物表面，长期吸取其汁液，但短时间内不弄死植物的（如蚜虫、菟丝子），算作**寄生**。
        

### ⚠️ 易错点四：把“捕食”曲线的上下位置看反

- **坑点**：在捕食曲线中，错把数量少的曲线当成被捕食者。
    
- **纠正**：根据能量金字塔，营养级越高，能量越少，数量通常也越少。因此，**数量多、先增加、先减少**的是被捕食者（如物像偏高的那条线）；**数量少、后增加、后减少**的是捕食者。
