
| **比较维度**  | **缩手反射 (Flexion reflex)**          | **膝跳反射 (Patellar reflex)**     |
| --------- | ---------------------------------- | ------------------------------ |
| **刺激原因**  | 针刺、烫伤等**伤害性刺激**                    | 敲击膝盖下方的**股四头肌肌腱**              |
| **突触数量**  | **多个突触**（通常 $\ge 2$ 个）             | **1个突触**                       |
| **神经元数量** | **3个**（感觉、**中间**、运动神经元）            | **2个**（仅感觉和运动神经元，**无中间神经元**）   |
| **反射弧类型** | **多突触反射**（复杂的多协同运动）                | **单突触反射**（最简单的反射弧）             |
| **反应表现**  | 多个关节联动（如屈肘、缩手、甚至大叫）                | 简单的股四头肌收缩，小腿前踢                 |
| **痛觉产生**  | **有痛觉**。虽然反射由脊髓控制，但信号会上传到大脑皮层产生痛觉。 | **无明显痛觉**。仅有被敲击的触觉或压觉，通常不产生痛觉。 |
