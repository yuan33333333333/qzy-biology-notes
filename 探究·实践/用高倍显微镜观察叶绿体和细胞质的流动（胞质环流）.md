

### 材料选择

- **藓类叶片**：**极薄，通常仅由一层细胞构成**。制作临时装片时不需要切片，可以直接观察。
    
- **菠菜叶（替代）**：必须取**稍带些叶肉的下表皮**。因为表皮细胞没有叶绿体，而靠近下表皮的叶肉细胞（海绵组织）细胞排列疏松、叶绿体大而少，便于观察。
    
- **黑藻**：叶片也是由单层或少数几层细胞组成。实验前需要**放在光照、室温条件下培养**，目的是**促进细胞呼吸和新陈代谢，使细胞质流动更明显**。


###  实验步骤
【准备工作】
黑藻提前进行**光照、温水培养**（唤醒细胞质流动）
  │
  ├───►【实验一：观察叶绿体】
  │     1. 取材：用**镊子**取藓类小叶（或菠菜叶带叶肉的下表皮）
  │     2. 制片：载玻片中央滴清水 ➔ 放叶片 ➔ 盖盖玻片（保持有水）
  │     3. 观察：**低倍镜寻找** ➔ 移至视野中央 ➔ **换高倍镜** ➔ 调细准焦螺旋
  │     4. 结果：看到绿色、扁平的椭球形或球形叶绿体
  │
  └───►【实验二：观察细胞质流动】
        1. 取材：从新鲜黑藻枝上取下一片【幼嫩】小叶
        2. 制片：**载玻片中央滴清水** ➔ 放**黑藻叶** ➔ 盖盖玻片
        3. 观察：**低倍镜找到叶肉细胞** ➔ **换高倍镜** ➔ 调细准焦螺旋
        4. 结果：以叶绿体为参照物，观察细胞质流动的方向和速度


### 考点

#### **【显微镜下的视错觉】**：
因为显微镜下看到的是**倒像**（上下左右颠倒），如果题目问你：“在显微镜下看到细胞质逆时针流动，实际的流动方向是什么？”

- **答案依然是：逆时针**。因为一个图形在平面上旋转 $180^\circ$ 后，顺时针依然是顺时针，逆时针依然是逆时针。这个结论可以直接死记硬背，高考屡试不爽。

#### 细胞质流动方向的“同与不同”

**在相邻的两个细胞之间：** 它们的流动方向**不一定相同**。可能细胞 A 是顺时针，紧挨着的细胞 B 却是逆时针。所以题目如果问“**所有细胞的流动方向是否一致**”，答案通常是**否定的**。

#### 影响细胞质流动速度的因素

细胞质的流动是一种**生命现象**，受细胞代谢强度的影响。高考常考它的影响因素：

- **温度**：在一定范围内，温度升高，酶活性增强，呼吸作用加快（提供能量 $ATP$ 变多），**细胞质流动变快**。但超过一定温度，细胞死亡，流动停止。
- **光照**：光照能促进光合作用和代谢，**流动变快**（这也是为什么黑藻实验前要照光）。
- **水分/化学物质**：细胞轻微失水或遇到某些毒性物质时，代谢受阻，**流动变慢甚至停止**。
