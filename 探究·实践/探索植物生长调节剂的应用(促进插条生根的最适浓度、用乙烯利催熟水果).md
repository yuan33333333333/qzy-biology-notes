# 1. 探索生长素类调节剂促进插条生根的最适浓度

## 实验前
### 理论依据
#### #预实验
>在进行科学研究时，有时需要在正式实验前先做一个预实验。这
>样可以为进一步的实验摸索条件，也可以检验实验设计的科学性和可行性，
>例如，在探索生长素类调节剂促进插条生根的最适浓度时，
>通过预实验**确定有效浓度的大致范围**，**可为确定最适浓度打下基础**。
>预实验也必须像正式实验一样认真进行才有意义。

### 材料用具
>生长旺盛的一年生枝 条，或者你们小组想要研究的其他植物的枝 条；蒸馏水；花盆；细沙；常用的生长素类 调 节 剂：NAA、2，4 -D、 吲哚丙酸、IBA， 可选其中的一种


## 实验中
【选材准备】 ──> 选取生长旺盛的**一年生枝条**（生长素多，形成层活跃）
   │
【预 实 验】 ──> 目的：为正式实验摸索条件，检验可靠性，减少盲目性
   │          配置梯度较大的系列溶液（如 0, 10⁻⁶, 10⁻⁵, 10⁻⁴, 10⁻³ mol/L）
   │
【正式实验】 ──> 缩小浓度梯度：在预实验中促进生根效果最好的浓度附近缩小范围
   │
【处理插条】 ──> 方法：【**浸泡法**】（低浓度长时间） 或 【**沾蘸法**】（高浓度短时间）
   │          ※ 注意：基部形态学下端朝下，深度一致
   │
【培养记录】 ──> **严格控制无关变量**（光照、温度、湿度、处理时间一致）
   │          计数因变量：观察并记录【生根数量】或【根的平均长度】
   │
【结果分析】 ──> 绘图：横坐标（调节剂浓度），纵坐标（生根数/根长度）
              分析：找出曲线最高点或对称点中间的浓度 ──> 锁定【最适浓度】

## 总结和其他考点
### 考点一：实验变量的精准控制

- **自变量：** **生长素类调节剂的浓度**。通常会设置一系列的浓度梯度，同时**必须设置一个用蒸馏水处理的对照组**（空白对照）。
- **因变量：** **插条生根的数量** 或 **根的平均长度**。
- **无关变量：** 处理的时间、植物材料的一致性（如枝条的长度、芽的数量、留叶面积、长势）、培养的环境条件（温度、湿度、光照）等。**无关变量必须保持相同且适宜。**
 
 枝条上的芽能够产生内源生长素。为了严格控制无关变量，每组插条上留有的**芽的数量必须相同**。如果芽太多，自身产生的生长素会对实验结果造成严重的干扰。

### 考点二：预实验的内涵与作用
教材中特别提到了“预实验”，这是高考考查实验探究能力时的最爱：
1. **为什么要做预实验？**
    如果对所研究植物的适宜浓度一无所知，直接做实验可能会因为浓度过高导致植物死亡，或浓度过低毫无效果，造成人力物力的浪费。预实验可以**为正式实验摸索条件，检验实验设计的科学性和可行性，免走弯路。**
2. **预实验的特点：** 浓度梯度**大**，组别**少**。
### 考点三：插条的处理方法对比

教材提到了两种处理方法，考试常以填空或选择题形式对比考查：

| **处理方法** | **药液浓度** | **处理时间**      | **操 作 要 点**                  | **适用场景**     |
| -------- | -------- | ------------- | ---------------------------- | ------------ |
| **浸泡法**  | 较低       | 较长（几小时至一天）    | 遮阴、空气湿度较高处进行，基部浸泡深约 **3 cm** | 适合实验室或精细操作   |
| **沾蘸法**  | 较高       | 极短（约 **5 s**） | 基部在药液中沾蘸深约 **1.5 cm** 即可     | 适合生产实践、大规模扦插 |
1. 无论哪种方法，处理的都是插条的**基部**（即形态学下端），绝不能颠倒倒插，因为生长素在植物体内进行的是**极性运输**（只能从形态学上端运输到形态学下端）。
2. 对于同一个实验，**所有浓度的方法必须一致，不能低浓度用浸泡法、高浓度用沾蘸法。**





# 2. 尝试利用乙烯利催熟水果

## 实验中

[材料准备] ──> 选取接近成熟但未完全成熟的水果（如绿香蕉、未熟番茄）
   │
[配制药液] ──> 注意安全防护（戴手套、通风、防火）
   │          利用**乙烯利在 pH > 3.5 时分解释放乙烯的特性**
   │
[实验分组] ──> 实验组：用适宜浓度的乙烯利溶液浸泡或喷洒水果
   │          对照组：用等量的蒸馏水处理
   │
[密闭留存] ──> 放入密封袋或容器中（防止乙烯散失，并诱导水果自身产生乙烯）
   │
[观察记录] ──> 观察水果的颜色、硬度、气味变化，对比成熟速度


## 总结和其他考点
### 乙烯利催熟水果的机理与安全

1. **工作机理：** * 乙烯利本身是一种植物生长调节剂（液体），在 **pH < 3.5** 时稳定。
    
    - 当它进入植物组织（pH 升高）后，会**分解释放出乙烯气体**。
        
    - 乙烯气体促进水果成熟，并能触发植物的**正反馈调节**，诱导水果自身产生更多乙烯，从而加速成熟。
        
2. **实验安全（非生物核心考点，但属于实验常识）：** 乙烯利具有强酸性和刺激性，必须在**通风良好**的环境下操作，注意防酸蚀和**防火**（遇到明火可燃）。
