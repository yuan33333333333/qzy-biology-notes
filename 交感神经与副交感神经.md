## 协调作用

| 器官        | 交感神经                             | 副交感神经                             |
| --------- | -------------------------------- | --------------------------------- |
| 心脏        | ↑心率                              | ↓心率                               |
| 瞳孔        | 扩大                               | 缩小                                |
| 支气管       | 舒张                               | 收缩                                |
| 胃肠        | 抑制                               | 促进                                |
| **胰岛**    | **使胰岛 A 细胞分泌胰高血糖素**，使得血糖含量**上升** | 刺激胰岛 B 细胞大量分泌**胰岛素**，使得血糖含量**下降** |
| 唾液腺       | 减少分泌                             | 增加分泌                              |
| **膀胱**    | **不会导致膀胱缩小**                     | **使膀胱缩小**                         |
| **血管**    | **收缩**                           | **无作用**                           |
| **肾上腺髓质** | **促进分泌肾上腺素**                     | **无作用**                           |
| **汗腺**    | **促进排汗**                         | **无作用**                           |
| **立毛肌**   | **收缩**                           | **无作用**                           |
